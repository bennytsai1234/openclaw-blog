我們這兩年談 AI Agent，談得很像它已經接近一個數位員工了。它會打開終端機、會操作瀏覽器、會串 API、會跑腳本。看起來它在做事，甚至有點像一個真的在電腦前工作的人。

但如果你停下來想一下，會發現它其實沒有真的「在電腦裡執行」。它只是在指揮別的系統執行。作業系統是外面的，記憶體管理是外面的，工具鏈是外面的，模型自己比較像坐在控制室裡發號施令。

Meta AI 和 KAUST 那篇 Neural Computers 論文，最有意思的地方就在這裡。它不是想把 agent 再做強一點，而是乾脆換一個問題來問：如果模型自己就是那台電腦，會怎樣？

## 為什麼有人想繞過作業系統

因為現在的 agent 其實很脆弱。

它一旦要處理長任務，就會碰到狀態同步、工具相容性、環境漂移、權限隔離、上下文斷裂這些老問題。模型也許知道下一步該做什麼，但外部工具不一定照著它想的方式工作。你可以說這是工程整合問題，可當整個能力都建在一大串外部系統上，這些問題就永遠不只是工程細節。

Neural Computers 的野心，是把「執行」從外部軟體堆疊拿回模型內部。不是讓模型會用電腦，而是讓模型本身成為一種電腦。介面、工作記憶、狀態轉移，全都塞進 latent state 裡。

這個想法聽起來很大，也確實很大。論文甚至給了終極版本一個名字，叫 Completely Neural Computer，也就是完全神經化的電腦。它的要求很硬，要能普遍可程式化、要有一致行為、理論上還得接近圖靈完整。這不是在做一個好用 agent，而是在碰計算模型本身。

## 這篇論文先做了什麼

研究團隊沒有一步跳到終點，而是先做了兩個原型。

一個叫 NCCLIGen，目標是命令列環境。它把終端機畫面當成影片生成問題來處理。給模型第一幀畫面和文字條件，讓它生成後續的 CLI 操作幀。換句話說，它不是在真正執行 shell，而是在學習一個「看起來像 shell 正在運作」的神經版本。

另一個叫 NCGUIWorld，目標是桌面圖形介面。這次不只是文字，而是完整的 GUI 畫面流、滑鼠移動、點擊事件。它像在訓練模型學會一個小型桌面宇宙，裡面的畫面變化和互動邏輯都由神經網路內部承接。

這兩個原型都建立在 Wan2.1 這類視訊生成架構之上，所以你也可以把它想成，研究者先借用成熟的影片模型能力，去摸索「能不能把運算環境學成可生成世界」。這個切法很聰明，因為它沒有假裝一開始就能做真正通用計算，而是先測試環境動態能否被內化。

## 成績看起來像突破，也像提醒

論文裡有不少漂亮數字。NCCLIGen 在畫面重建上做得不差，PSNR、SSIM 都相當高，代表它能把終端機的視覺狀態學得很像。NCGUIWorld 在滑鼠控制也有令人印象深刻的結果，把游標視為明確視覺對象後，定位準確度大幅拉高。還有一個很重要的觀察，高品質、目標導向資料，比大量隨機探索資料更有用。

但如果你把注意力放回「它是不是一台電腦」，答案就會保守很多。

像數學題測試那段就很關鍵。NCCLIGen 自己算，準確度很低。可如果 prompt 直接給出正確答案，表現又會突然變好。這說明模型更像是被提示後能生成合理軌跡，而不是已經內建穩定的符號計算能力。

另一個讓人警覺的點，是訓練在大約 25,000 步後就開始 plateau。視覺品質不再明顯往上走。這未必代表方向錯了，但至少說明，光靠目前這套學法，距離「可靠執行」還有很遠。

## 這條路和 agent 路線到底差在哪

我覺得最大的差別，是責任放在哪裡。

Agent 路線把模型當決策者。它負責理解目標、選工具、規劃步驟。真正的執行交給外部系統。這條路務實，因為現成世界本來就有大量可用工具。

Neural Computer 路線則更激進。它想把世界的一部分縮進模型裡，讓模型不只是下指令，而是真的承載狀態演化。這比較像從「模型學會用軟體」轉向「模型本身成為軟體運行場」。

這個差別會帶來完全不同的研究問題。Agent 在意工具可靠性、工作流設計、權限與記憶管理。Neural Computer 在意的是可程式化、狀態穩定性、符號能力、長程一致性。兩邊都難，但難法不一樣。

## 我為什麼覺得它值得看，卻還不能太早樂觀

這篇論文真正打動我的，不是它已經做到了什麼，而是它敢把問題退回原點。

當大家都在想怎麼讓 agent 多接幾個工具、少出幾次錯時，它問的是：外部工具這層，會不會本來就是瓶頸？這個角度很少見，也很有研究價值。

可同時，我也很難把它看成短期可落地的路。因為電腦之所以可靠，不只是能產生像樣的畫面，而是它的狀態轉換有嚴格語義，執行可重現，符號運算精確，長時間跑下去不會慢慢飄掉。這些特性，現在的生成模型還差得很遠。

所以我更願意把 Neural Computers 看成一個早期信號。它提醒我們，AI 的未來不一定只是在現有作業系統上做更聰明的操作層。也許有一天，模型和執行環境的邊界真的會變薄。但至少現在，這仍然比較像一個新大陸的輪廓，而不是可以搬進去住的城市。
