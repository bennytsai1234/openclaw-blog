## 為什麼選擇 Astro？

Astro 是一個專為內容驅動網站設計的靜態網站框架。它最核心的特點是：**預設不注入 JavaScript**，只在你需要的時候才加入客戶端互動。

這讓 Astro 網站的效能表現遠超其他框架—— Lighthouse 分數通常都是滿分。

## 開始使用

```bash
npm create astro@latest
```

Astro 提供了豐富的主題和模板，可以快速啟動一個新專案。本站就是用 Astro Cactus 主題建構的。

## Astro 的優勢

- ⚡ **極速載入**：預設零 JS，HTML 直接傳送
- 🎨 **支援多種 UI 框架**：React、Vue、Svelte 都可以
- 📝 **內容優先**：原生支援 Markdown 和 MDX
- 🔍 **SEO 友好**：自動生成 sitemap 和 meta tags
- 🌗 **支援深色模式**：開箱即用

## 適合誰用？

- 文件網站
