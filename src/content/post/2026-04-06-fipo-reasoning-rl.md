AI 模型在解決複雜推理任務時，為何常常在接近終點時戛然而止？答案藏在強化學習（RL）的獎勵機制裡。在現行機制下，寫下一個無關緊要的逗號，與找到扭轉全局的關鍵線索，所獲獎勵完全相同。當每個思考步驟的價值被無情地平均化，AI 便失去了「深思熟慮」的動機，導致推理能力難以突破瓶頸。

### 獎勵機制出了什麼問題？

問題核心在於獎勵分配。當使用 GRPO 等方法訓練模型時，系統會根據答案對錯給予二元獎勵：答對為 1，答錯為 0。這得來不易的 1 分，會被平均分給思維鏈中的「每一個」token。這意味著，一個連接詞「然後」，和那句點出矛盾的「神來之筆」，在模型眼中價值相等。

在這種機制下，模型學到的是一種保守策略。既然多思考一步並不保證更高的「單位獎勵」，反而可能因鏈條變長而增加犯錯風險，最理性的選擇就是「淺嘗輒止」。這就是為何許多模型的推理鏈長度達到某個瓶頸後便難有寸進。它們並非不能想得更深，而是訓練機制未給予它們這樣做的理由。

### 未來影響力，如何衡量想法的價值？

過去，研究者試圖引入獨立的「價值模型」來評估每一步的長期價值。但這種模型不僅需要大量高品質數據預訓練，其成效也很難與演算法本身的效果清晰剝離。

Qwen 團隊的 FIPO（Future-KL Influenced Policy Optimization）提供了一個新視角：無需外部價值模型。FIPO 的核心洞見是將獎勵焦點從「評估當前步驟的內在價值」轉移到「衡量該步驟對未來的影響力」。它不去問「這個 token 值多少？」，而是問「生成這個 token 後，模型後續的預測發生了多大改變？」

這個「改變」透過 KL 散度（Kullback–Leibler divergence）來度量。FIPO 會計算並比較兩組機率分佈的差異：一組是「生成某 token *之前*，模型對後續所有 token 的預測」，另一組是「生成該 token *之後*，模型對後續所有 token 的預測」。

若一個 token 的出現，能大幅提升模型對正確推理路徑上後續詞語的預測信心，使模糊的未來變得清晰，其 KL 散度就會很大，代表它產生了巨大的「未來影響力」，理應獲得高額獎勵。反之，若一個 token 讓後續推理更混亂，它就會受到懲罰。此計算過程完全「自給自足」，只需比較模型自身在不同時間點的狀態即可。

### 如何約束只看遠方的模型？

然而，一個完全由「未來影響力」驅動的模型，容易因過度追求能引發劇烈未來變化的 token 而導致訓練崩潰。模型會產生巨大偏離（drift），最終喪失思考能力。

為了解決此問題，FIPO 設計了兩套穩定機制：

1.  **折現因子（Discount factor）**：預測越遠的未來，不確定性越高。因此，一個 token 對其鄰近 token 的影響權重，應高於對遠處 token 的影響。這讓模型更關注近期影響力，避免不切實際的預測。

2.  **極端值過濾（Extreme value filtering）**：一道安全閥。當 FIPO 偵測到模型在相鄰訓練步驟間的策略分佈發生過於劇烈的變化時，會直接忽略此訓練樣本，避免因策略突變導致的災難性後果。

這兩套機制確保了 FIPO 在追求「深度思考」的同時，能在探索與穩定之間達成平衡。

### 一個聰明的答案，但它的應用範圍為何？

FIPO 的實驗結果令人振奮。研究團隊在一台未接觸過合成長思維鏈數據的「純淨」模型上，僅用公開數據集訓練，就在高難度數學競賽 AIME-2024 基準測試中，讓模型的平均推理鏈長度幾乎翻倍，表現與頂尖推理模型相當。這證明 FIPO 確實能打破「長度天花板」。

FIPO 最大的貢獻，是準確指認了「均勻獎勵分配」與「推理鏈停滯」的因果關係。問題根源並非演算法疏忽，而是「任務結構的假設」出了錯。我們過去下意識地假設「每個推理步驟價值相等」，但對需要深度探索的推理任務而言，此假設從一開始就站不住腳。

當然，FIPO 在 AIME 數學競賽上的成功，很大程度是因為數學問題存在客觀、唯一的正確答案，讓「未來影響力」的計算有清晰目標。但對於沒有客觀標準的開放式推理任務，例如構思原創故事結局，FIPO 定義的「未來影響力」是否依然有效，仍有待驗證。

儘管如此，FIPO 的出現標誌著我們在訓練 AI 思考的路上邁出了重要一步。它啟示我們，要打造更聰明的思考者，首先需要學會如何更聰明地獎勵「思考」本身。
