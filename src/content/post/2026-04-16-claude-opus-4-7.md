2026 年 4 月 16 日，Anthropic 正式發布 **Claude Opus 4.7**，這是 Claude Opus 4.6 的增量升級版本。這次發布值得關注的亮點在於：它不僅在軟體工程能力上有所突破，還成為首個內建自動化網路安全防護機制的公開模型。

## 軟體工程能力的顯著提升

根據 Anthropic 官方公告，Opus 4.7 在**高級軟體工程任務**上比 Opus 4.6 有明顯進步，尤其在最困難的任務上表現更佳。早期的測試者回饋表示，現在可以放心地把最具挑戰性的程式碼工作交給 Opus 4.7 處理，無需像以前那樣 close supervision。

這款模型能夠處理複雜的長期任務，保持嚴謹和一致性，精準注意指令，並且會在回報結果前自行驗證輸出。

## 視覺能力的提升

Opus 4.7 的視覺能力也大幅提升，能以更高的解析度處理圖片。這對於需要分析圖表、介面設計或文件掃描的應用場景非常有幫助。

此外，Opus 4.7 在完成專業任務時更加有品味和創意，能產出更高品質的介面、簡報和文件。

## 與 Claude Mythos 的定位差異

值得注意的是，Anthropic 坦誠表示 Opus 4.7 **比他們最強大的模型 Claude Mythos Preview 能力較弱**。上週 Anthropic 宣布了 Project Glasswing，強調 AI 模型在網路安全方面的風險與效益。他們決定先在較低能力的模型上測試新的網路安全防護機制，Opus 4.7 就是首個實作這些防護的公開模型。

Opus 4.7 的網路安全能力不像 Mythos Preview 那樣先進（事實上在訓練過程中他們曾嘗試刻意降低這些能力）。這款模型會自動偵測並阻擋涉及禁用或高風險網路安全用途的請求。

## Cyber Verification Program

為了讓安全研究人員能夠合法使用 Opus 4.7 進行網路安全相關工作（如漏洞研究、滲透測試、紅隊演練），Anthropic 推出了新的 **Cyber Verification Program**，符合資格的安全專業人員可以透過表單申請加入。

## 價格與可用性

Opus 4.7 價格維持與 Opus 4.6 相同：
- 輸入：每百萬 token 5 美元
- 輸出：每百萬 token 25 美元

目前已可在所有 Claude 產品、Anthropic API、Amazon Bedrock、Google Cloud Vertex AI 和 Microsoft Foundry 上使用。開發者可以透過 Claude API 使用 `claude-opus-4-7` 模型。

## 早期測試回饋

根據早期訪問測試者的回饋，Opus 4.7 在規劃階段就能發現自己的邏輯錯誤，並在執行過程中加速進行，這種速度與精度的結合對大型金融科技平台來說可能是革命性的突破。有測試者表示，Opus 4.7 在處理真實世界的非同步工作流程（自動化、CI/CD、長期任務）方面表現出色，不僅僅是原始能力強大。

## 觀點

Opus 4.7 的發布策略反映了 AI 產業一個趨勢：頂尖模型與公開模型之間的差距正在擴大。Anthropic 選擇將最強大的 Mythos 留給網路安全合作夥伴測試，而讓一般用戶使用經過安全加固的 Opus 4.7，這種做法或許會成為未來 AI 安全的標竿——先在較弱的模型上驗證安全機制，再逐步擴展到更強大的模型。

對於開發者而言，Opus 4.7 在軟體工程方面的提升是最實質的亮點。如果你需要一個能 handle 複雜coding任務、又能自己驗證輸出的助手，這次升級值得一試。
