## 今日觀察

5 月 17 日的 AI 開發者生態迎來三個實體進展：OpenAI 為 Codex 推送大型效能更新，宣稱大型倉庫的 Git 操作提速 10 至 50 倍；Vercel Labs 低調推出一款專為 AI Agent 設計的實驗性程式語言 Zero；而 OpenAI 則與馬耳他政府達成協議，讓完成 AI 素養課程的公民免費使用一年 ChatGPT Plus。三件事各有不同層面的意義，但共同指向同一個趨勢：AI 工具正在從「能用」走向「好用」，從「個人體驗」走向「制度性部署」。

---

## 主題一 — Codex 效能更新：從「能跑」到「跑得快」

Codex 這次更新有兩個亮點值得注意：一是效能，二是產品化深度。

效能面上，OpenAI 宣稱大型倉庫中 Git 操作（提交、推送、分支、PR 建立）提速約 10 至 50 倍，切換執行緒時重繪製降低約 75%。對日常使用小型專案的開發者而言，這個數字可能感受不深；但對處理數十萬行程式碼、依賴大量 Git 歷史的團隊而言，50 倍的差距可能就是「等三十秒」和「等半秒」的分界。根據 OpenAI 開發者帳號的公告，這次更新很大程度是回應社群的抱怨——過去 Codex 的 Git 功能在大型 Monorepo 底下幾乎處於半廢棄狀態。

產品化深度體現在自訂鍵盤快捷鍵的開放。這不是什麼了不起的功能，但釋放了一個明確訊號：Codex 正在從「雲端 AI 助手」往「開發者本地工作站」的方向移動。快捷鍵自訂代表它允許開發者把 Codex 整合進自己的編輯習慣，而不是要求開發者去適應一個固定工作流。與此同時，Git 關鍵操作被移回審查流程（Review Flow），讓流程更易觸及——這也是對企業用戶的直接示好。

值得觀察的是，幾乎同一時間社群傳出 GPT-5.5 效能下降的問題。Codex 團隊成員 Tibo 隨後確認已修復問題並重置所有付費使用額度。這說明 Codex 仍在高度迭代階段，穩定性還沒有達到企業級產品的標準，但修復速度算是及時。對依賴 Codex 做生產開發的團隊來說，這種「發現問題就修、再送你額度」的補償模式，算是可以接受的回應。

## 主題二 — Vercel 推出 Zero：為 Agent 而生的實驗性語言

Vercel Labs 默默在 GitHub 上線了一個新專案：Zero，一款標榜為 AI Agent 打造的程式語言。官方形容它是「系統級語言，用於構建更小、更快的原生工具」，主打显式能力（explicit effects）、可預測記憶體以及結構化的編譯器輸出。

這件事為什麼值得注意？因為目前主流的 Agent 工具鏈——無論是 MCP（Model Context Protocol）還是各種 function calling 機制——都是建立在既有程式語言之上的外層抽象。開發者用 Python 或 JavaScript 包裝工具，讓語言模型呼叫。Zero 的做法不同：它試圖從語言本身解決問題，把 Agent 需要的特性（能力邊界明確、記憶體可預測、編譯器輸出結構化）內建進語法層次。

根據 GitHub 上的說明，Zero 目前處於高度實驗階段，語言本身尚未穩定。官方的態度很誠實：歡迎開發者試用與回饋，但不建議拿去做生產專案。目前已提供安裝腳本、VSCode 語法擴充，以及文件網站。從開源策略來看，Vercel 這次選擇先社群後商業的節奏，算是符合他們一貫的工程師文化。

不過必須說清楚的是：Zero 目前只是「有可能」改變 Agent 工具鏈的東西，還不是「已經」改變了什麼。這個階段入場，是對的方向，但對多數開發者而言，現在更實際的做法是保持關注，而不是立刻跳下去學。

## 主題三 — OpenAI 與馬耳他：AI 民主化的國家尺度實驗

OpenAI 宣布與馬耳他政府合作，向完成當地 AI 素養課程的公民提供一年免費 ChatGPT Plus。這是「OpenAI for Countries」計畫的一部分，馬耳他是目前唯一進入這種合作模式的主權國家。

具體機制是：馬耳他公民先完成由馬耳他大學設計的「AI for All」課程，內容涵蓋 AI 基礎概念、能力與限制、以及負責任的使用方式。完成後由馬耳他數位創新局（MDIA）分發帳號，領取為期一年的 ChatGPT Plus。這個模式與一般「政府發補貼讓民眾用 AI」的邏輯不同，它把教育當作門檻，而不是直接發帳號。

馬耳他之所以成為第一個這種規模的試點，有其地緣背景：人口僅約 53 萬，數位治理基礎相對完善，且政府對科技創新態度積極。OpenAI 的「各國計畫」負責人 George Osborne 明確說，這種模式「未來會推廣到其他國家」，但目前各國的進度差異很大——OpenAI 已在與愛沙尼亞、希臘等國洽談教育系統相關的合作。

這件事對開發者的啟示有兩層：第一，AI 素養教育正在從「個人選擇」升級為「國家政策」，對接下來的 AI 產品滲透率會有長期影響；第二，「ChatGPT Plus 作為公共資源」的モデル一旦規模化，對付費 AI 產品的定價策略會帶來壓力。馬耳他模式若成功，其他小型發達經濟體很可能會跟進。

## 其他值得關注

- **微信讀書 AI 助手 Skill**：騰訊旗下微信讀書推出官方 Skill，用戶連接帳號後可讓 AI 搜書、匯出筆記與閱讀數據。這是騰訊系產品中少見的官方 AI 助理整合，顯示微信讀書正在把 AI 助手定位成閱讀流程的一部分，而非獨立的AI功能。對繁體中文閱讀市場的影響有待觀察。

- **上海電信上線 Token 算力服務**：上海電信推出多檔大模型 API 調用量套餐，1 元可兌換 25 萬 Token，並限量提供 2500 萬 Token 免費體驗。這個動作代表中國電信營運商正式進入模型 API 分銷市場，結合話費支付的機制降低付費門檻，對中小型開發者有一定吸引力。
