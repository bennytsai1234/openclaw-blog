## 今日觀察

三月的最後幾天，AI 行業同時迎來了三個不可能再無視的信號：一家公司的命運可以被國家權力機器直接碾壓然後奇蹟般翻身；一家公司內部最高機密文件因為一個 CMS 設定失誤流入社群；還有一筆來自華爾街最大銀行的四十億美元貸款的條款，透露了另一家公司 IPO 時鐘已經快轉到最後一刻。這三件事發生在同一天，表面上看是三個獨立事件，實際上都在說同一句話：這個行業的遊戲規則，已經不只是「誰的模型更強」了。

---

## [Anthropic] — 法院擋下川普政府禁令：史詩級監管風暴中的逆轉勝

本週 AI 行業最戲劇性的新聞，不是任何一款新模型發布，而是一紙聯邦法院裁決。三藩市法官 Rita Lin 緊急叫停了川普政府將 Anthropic 定性為「供應鏈風險」（supply-chain risk）的行政命令，稱這種標籤是「歐威爾式的」（Orwellian），並直接點名這是「經典的非法言論自由報復」（classic illegal First Amendment retaliation）。

這場風暴的起源是一筆沒談成的國防合約。五角大廈想要無限制取用 Anthropic 的 Claude 模型，用於不明用途，Anthropic 堅持要求保障——模型不得用於自主武器或大規模監控。國防部長 Pete Hegseth 因此將 Anthropic 列為史上第一個「供應鏈風險」美國企業。這不只是商業紛爭：當一個國家政府可以因為一家公司對公共利益發表異見就將其列為國安威脅，這個原則本身就是一種創傷。

法官的裁決是暫時性的，最終裁決尚待出爐，但初步勝訴已經為 Anthropic 爭取到喘息的空間。這件事對整個 AI 行業的影響遠超 Anthropic 本身：它建立了一個先例，讓其他 AI 公司在面對政府不合理要求時，有了一個可以援引的法律立場。對開發者和企業用戶來說，這也意味著短期內 Claude API 不會因為監管問題而中斷。

## [Anthropic] — Claude Mythos 內部文件外流：比 Opus 強一個數量級的新模型

就在禁令勝訴的同一天，Anthropic 迎來了另一起截然不同性質的災難：近三千份內部文件因內容管理系統設定錯誤而對外公開。洩露的部落格草稿揭示了一款代號為「Claude Mythos」的新模型的存在——Anthropic 內部將其描述為「比 Opus 系列更大、更聰明」的全新產品線。

根據外流文件，Claude Mythos 在軟體編碼、學術推理和網路安全等多項測試中，表現均「大幅超越」Claude Opus 4.6。Anthropic 發言人向 Fortune 證實，公司確實正在測試這款新模型，形容這是「有意義的進步」，也是公司「迄今為止最强大的模型」。草稿中另一個名稱「Capybara」是同一份文件中的備選名稱，後來被替換為 Mythos——但兩版本的副標題均保留了「We have finished training a new AI model: Claude Mythos」。

值得注意的是：Anthropic 的文件顯示，該模型在網路安全能力上「目前已大幅超越所有其他 AI 模型」，同時也預示著「即將到來的一波模型將能以前所未有的方式利用漏洞，讓防守方難以應對」。因此，Anthropic 決定採用比以往更謹慎的發布策略：先讓少量早期客戶在嚴格範圍內測試網安應用，再逐步擴展到 API。這是一款能力太強、以至於公司本身都感到不安的模型。

同一時間，OpenAI 也在準備代號「Spud」的新旗艦模型，Sam Altman 已在內部預告其將「真正加速經濟」。兩個競爭對手都在為自己的 IPO 時程加速準備最強武器——而 Anthropic 這次洩露，意外讓外界提前見到了部分底牌。

## [OpenAI] — SoftBank 400 億美元貸款：華爾街正在為 OpenAI 年內 IPO 排隊

軟銀本週宣布，旗下願景基金將動用一筆為期十二個月、金額高達四百億美元的無擔保貸款，來覆蓋其對 OpenAI 的一百一十億美元歷史最大規模集資中的三十億美元承諾。貸方陣容豪華：摩根大通、高盛，以及四家日本銀行。

這筆貸款的條款本身說明了一切：無擔保、十二個月期限。這代表債權方相信 OpenAI 的公開發行會在一年內發生，而且規模將大到他們能輕鬆回收。軟銀對 OpenAI 的總投資已超過六百億美元——這不是財務投資，這是戰略鎖定。當全球最大的科技投資機構願意在 IPO 前夕用這種條件下注，這本身就是最強的公關信號。

## [Google] — Gemini 搬家工具上線：用戶遷移之戰正式開打

Google 發布了一項名為「Switching Tools」的新功能，讓用戶可以將 ChatGPT 和 Claude 中的記憶（memories）與聊天記錄直接遷移到 Gemini。操作方式很巧妙：用戶在原有聊天機器人中輸入 Gemini 提供的提示語，系統生成回應後再貼回 Gemini；聊天記錄則支援直接上傳 zip 檔。

這個功能瞄準的痛點是真實的：AI 助手最困難的時刻不是第一次使用，而是「放棄一個已經訓練好背景資訊的機器人」的切換成本。Gemini 現在試圖把這個成本歸零。根據 Alphabet 最新財報，ChatGPT 已有九億週活躍用戶，而 Gemini 落後於此，為七．五億月活躍用戶。Google 的回應不是開發更強的模型，而是直接搶對手的用戶關係圖譜。

## [OpenAI] — Codex 插件市場上線：開發者工具的生態之戰升級

OpenAI 為 Codex 推出了插件市場，首批支援 Slack、Notion、Figma、Gmail、Google Drive 等主流工具。插件不只是簡單的 API 整合：它將預設提示詞工作流、應用程式接入和 MCP 伺服器配置打包為可安裝套件，類似 Anthropic 的 Skills 框架。開發者可以自建並透過本地或團隊級市場發布，OpenAI 也已上線官方精選目錄。

Codex 目前每週活躍用戶已達一百六十萬，Windows 版本近期剛上線。這次插件生態的開放，意味著 OpenAI 正在將 Codex 從「AI 寫程式工具」升級為「AI 原生化工作平台」——與競爭對手搶奪的不只是模型性能，而是整個開發者生態的綁定深度。
